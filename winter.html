Winter Classes

Do you want to learn coding and be a part of a good cause? Our winter classes are perfect for you! We partnered with Cradles to Crayons to create $5 coding classes with all of the proceeds going to Cradles to Crayons. Although you do not need to donate to attend classes, we strongly urge you to be a part of this cause.

Class Tracks

What will you learn?

In our python track you will learn basic coding concepts in python! Lessons include data types, arrays, dictionaries, functions, and so much more!

FAQ

All students from grades 3 to 12 are welcome to apply!

The camp is completely FREE.

Unfortunately we are unable to provide computers to students, so you will need your own computer with internet access. You can, however, also use a tablet or an iPad.

No prior experience necessary! Our camp is open for everyone and teaches the basics of coding in python.

Since python is a very large language with so much to learn, this class does not have a final project unfortunately. We keep students learning until the very last day, with mini projects in between!

What will you learn?

In our java track you will learn basic coding concepts in java! Lessons include data types, user input, arrays, methods, classes, and so much more!

FAQ

Any students from grades 3 to 12 are welcome to apply! However, if you are in grades 3 to 6 we encourage you to explore our other tracks first, as we found that java is a bit more complicated for younger students to understand.

The camp is completely FREE.

Unfortunately we are unable to provide computers to students, so you will need your own computer with internet access. You can, however, also use a tablet or an iPad.

No prior experience necessary! Our camp is open for everyone and teaches the basics of coding in java.

The final project for our java track is an online shop code that makes use of everything the students have learned!

What will you learn?

In our java track you will learn basic web development concepts using HTML, CSS, and Javascript! Lessons include creating outlines for website, coding a website in HTML, using CSS for style, and basic Javascript concepts!

FAQ

Any students from grades 3 to 12 are welcome to apply! However, if you are in grades 3 to 6 your curriculum might be slightly different than older students.

The camp is completely FREE.

Unfortunately we are unable to provide computers to students, so you will need your own computer with internet access. You can, however, also use a tablet or an iPad.

No prior experience necessary! Our camp is open for everyone and teaches the basics of web development.

The final project for our web development track is creating your own website! Using all the coding concepts you have learned, you will be able to make a multi-page website!
